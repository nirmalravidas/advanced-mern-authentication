# Advanced MERN Authentication

Advanced MERN stack authentication system with secure login, registration, password reset, email verification, and JWT-based authentication.

## Features
- User registration and login
- Password hashing with bcrypt
- JWT-based authentication and authorization
- Email verification for new users
- Password reset via email with token
- Role-based access control (future enhancement)
- Separate frontend and backend structure

---

## Tech Stack
- **Frontend:** React, Tailwind CSS
- **Backend:** Node.js, Express
- **Database:** MongoDB
- **Authentication:** JWT, bcrypt, email verification
